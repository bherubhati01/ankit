# Myinsta - A Social Media Django Web App

## Introduction
**Myinsta** is a social media web application built using Django, inspired by Instagram. It allows users to share posts, follow other users, like and comment on posts, and engage in real-time messaging. The platform provides a seamless experience with a responsive UI built using Tailwind CSS and JavaScript.

## Tech Stack
- **Frontend:** HTML, CSS, JavaScript, jQuery, Tailwind CSS
- **Backend:** Python, Django
- **Database:** SQLite, MySQL
